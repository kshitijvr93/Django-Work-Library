The git repository structure uses the
filesystem structure of the project
folder that was initialzed, plus it adds
some files to manage git functions.

You push your code from the local
repository to a remote repo (e.g.,
your own repositories or GitLab or
GitHub,or other services) and pull code
from the remote to your local repo.

On your local Windows PC, visit the topmost directory
of your project code.

Right Click and Select "Git Bash Here" to open a git bash window.
At the $, enter git init

You'll see a few lines of output that you can ignore and
the $ prompt will appear on a new line.
